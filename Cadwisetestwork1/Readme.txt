﻿Приложение написано под платформу .NET 6, однако с переносимостью до версии 4.5 не должно быть проблем.
Приложение состоит из одного окна "Парсер". В окне находится контейнер для задач.
В левом верхнем углу находится кнопка "открыть", которая вызывает окно выбора опций и файлов.
Для активирования кнопки старт необходимо выбрать входной и выходной файл, остальные опции - по усмотрению.
